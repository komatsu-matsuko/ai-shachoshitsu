# フォルダとファイルの説明（日本語）

このドキュメントでは、**どこで何が行われているか**「**何がどこに書いてあるか**」「**何のためのフォルダ・ファイルか**」を日本語で整理しています。

---

## プロジェクトの一番上（AI社長室フォルダ直下）

| ファイル名 | 何が書いてあるか | 何のためか |
|------------|------------------|------------|
| **README.md** | 社長室の目的・使い方・コマンド・毎日の流れ | 最初に読む「使い方」の説明書 |
| **運用仕様書.md** | 組織図・役割・営業構造・MTG・評価・CEO思考など | 運用の正式な仕様（詳細版） |
| **フォルダとファイルの説明.md** | この一覧（どこに何があるか） | フォルダ・ファイルを探すときの地図 |
| **埋め方ガイド.md** | どの順で・何を書けばよいか | 中身を埋めるときの手順とチェックリスト |

---

## scripts（自動化スクリプト）

**何のためか:** 日次売上（Uber・エアレジ・LINE）を **手入力せず** API で取得し、Google スプレッドシートに自動で 1 行追記する。

| フォルダ／ファイル | 何が入っているか | 何のためか |
|--------------------|------------------|------------|
| **daily_sales_to_sheets/** | Python スクリプト（エアレジ・Uber・LINE 取得 → Sheets 書き込み） | 毎日 1 回実行すると日次売上報告が自動で埋まる |
| **README_automation.md** | セットアップ・実行・タスクスケジューラのやり方 | 社員 PC や Cowork で動かす手順 |

認証情報は .env に記入（Notion で管理）。.env は git に含めない。

---

## .cursor / rules（AIの行動ルール）

**何のためか:** Claude（AI）がこのプロジェクトで話すときに、**必ず読んで従うルール**が入っています。中身を変えると、AIの動き方が変わります。

| ファイル名 | 何が書いてあるか | 何のためか |
|------------|------------------|------------|
| **ai-executive-office.mdc** | AI参謀団の人格・表示の仕方・コマンドの意味・人間組織（笹川・岡・吉浦・濱村・結城） | 「社長室としてどう振る舞うか」の最上位ルール |
| **ai-memory-system.mdc** | 会社メモリ（company-memory）を「長期記憶」としてどう読むか・いつどのフォルダを見るか | 戦略や判断の前に、**会社の情報を読んでから答える**ようにするため |
| **ai-thinking-framework.mdc** | 答え方の型（洞察→推奨→実行計画） | 雑談ではなく**経営型**で答えるため |

## 英語のファイル名について

- **AIは日本語のファイル名・フォルダ名を問題なく扱えます。** 英語である必要はありません。
- 現在は、ツールやルールとの互換性を考えて、フォルダ名・ファイル名の多くを英語のままにしています。
- **中身はすべて日本語で書いて大丈夫**です。むしろ、会社のことを書くなら日本語の方が分かりやすいです。
- 今後、フォルダ名・ファイル名を日本語に変更することも可能です。その場合は、`.cursor/rules` 内の参照パスも一緒に書き換える必要があります。

## まとめ：どこで何をするか

| やりたいこと | どこを見る・どこに書く |
|--------------|--------------------------|
| 社長室の使い方を知りたい | README.md |
| 運用の正式な仕様を確認したい | 運用仕様書.md |
| どのファイルが何か分からなくなった | フォルダとファイルの説明.md（このファイル） |
| AIの動き方を変えたい | .cursor/rules 内の .mdc |
| AI役員の定義を変えたい | ai-office 内の .md |
| 会社の情報をAIに覚えさせたい | company-memory の該当フォルダに .md で追記 |
| 週次MTGのログを残したい | company-memory/09_meetings/weekly/ に日付のファイル |
| 月次レビューを残したい | company-memory/09_meetings/monthly/ に月のファイル |
| 社長ダッシュボードの元データを更新したい | company-memory/10_dashboard/ の3ファイル |
| CEOの考え方をAIに反映させたい | company-memory/08_ceo/ 特に ceo-thinking-framework.md |
